Последовательность работы проекта

Загрузка CSV-файла

CSV-файл Объекты спорта.csv размещается в папке resources/data

Файл читается с корректной кодировкой (UTF-8)

Парсинг данных

Из каждой строки CSV извлекаются:

наименование объекта спорта

регион

адрес

дата внесения в реестр

Москва и Московская область объединяются в один регион
«Москва и Московская область»

Сохранение данных

Регионы и объекты спорта сохраняются в базу данных SQLite

Для каждого региона ведётся подсчёт количества объектов спорта

Аналитика

Вычисляется среднее количество объектов спорта по регионам

Регионы сортируются по количеству объектов спорта

В консоль выводятся 3 региона с наибольшим количеством объектов

Построение гистограммы

Строится столбчатая диаграмма (гистограмма)

По оси X — регионы

По оси Y — количество объектов спорта

График визуализирует распределение объектов спорта по регионам
